Base Number Conversion

We are used to expressing values as numbers in the decimal number system, which can also be described as base-ten. However, there are as many bases as there are numbers to express values by, such as binary for computers or hexadecimal for colors.

In short, a base dictates both the range of numbers as well as the value of of each digit depending on it's position in the number. For example, base-ten has a range of ten numbers, from zero to nine. Additionally, the values of the digits vary depending if they are in the one's (100), ten's (101), or hundred's (102) places just to name a few.

Essentially, each place in a base represents that base number to a power, starting with zero as the power on the right, and incrementing by one. This is true for any base number. For example, with base-two, the place values from right to left go one (21), two (21), four (22), eight (23), etc. Though this covers the basis, this is a great source to master this concept.
